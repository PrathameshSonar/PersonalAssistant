# PersonalAssistant
A python based program for performing daily routine activities through a computer assistant.

Libraries required : 
    1)speechrecognition
    2)playsound
    3)gTTS
    4)PyAudio
    5)datetime
    6)imaplib
    7)smtplib
    

Run a assistant.py for execuation of code

Run cal_setup.py for setting up connection with google calender
